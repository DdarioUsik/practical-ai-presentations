## 1. КОМПАНИЯ

**Dr.Head** — премиальный аудио-ритейл, 20 лет на рынке.

### Масштаб
- 3 магазина в России (2 Москва, 1 Санкт-Петербург) + 1 в Дубае
- ~200 сотрудников всего
- ~50 офлайн-продавцов, ~25 онлайн-менеджеров
- Компания растёт, несмотря на падение рынка и конкурентов (-10-15%)

### Онлайн-присутствие
- Россия: @dr.head (Instagram, 48K подписчиков), doctorhead.ru, VK, Telegram (t.me/drheadpersonal, t.me/drheadCHAT), YouTube/Rutube (drheadtv, 576+ видео), TikTok, Yandex Zen
- Дубай: @drhead.ae (Instagram, 6.1K подписчиков)
- Телефон: 8-800-333-09-89

### IT-инфраструктура (текущая)
- **1С**: основная система учёта, ERP, бухгалтерия. Переведена на английский для Дубая. Есть дописанная «CRM» корявыми руками + Excel
- **Битрикс**: используется на 30-40%, в основном как таск-трекер + движок сайта
- **Tally**: бухгалтерская программа в Дубае, сертифицирована Эмиратами, синхронизирована с 1С (работает криво, двойной ввод данных)
- **Манго**: телефония, есть возможность настроить трекинг и сохранять записи звонков
- **Excel**: используется для множества процессов, данные разрозненные
- IT разделено на 3 части БЕЗ общего руководителя (CTO нет):
  1. Серверы, железо, сетевая инфраструктура
  2. Веб-разработка (фронт/бэк сайтов, сателлиты)
  3. Разработка/доработка модулей 1С, обмены данных

### Ключевая проблема
CRM нет. 3-4 попытки внедрения за 20 лет — ни одна не доведена до конца. Причины: отсутствие CTO, уход ответственных людей в процессе внедрения, смена приоритетов, кризисы.

## 2. УЧАСТНИКИ (стейкхолдеры)

- **Боль**: Не может масштабироваться с текущим хаосом. Хочет открываться в США, Лондоне, Германии, Саудовской Аравии, Абу-Даби — но понимает, что будет масштабировать хаос.

### Алексей — руководитель автоматизации

**Ближайший шаг**: Индивидуальные разборы с каждым ключевым участником (30-40 мин) → сборка общей карты → приоритеты → план.

- Цена: 150 000 руб (если удастся закрыть)

- Артём мягкий — не будет продавливать стратсессию команде

## 9. РЕЛЕВАНТНЫЕ КЕЙСЫ PRACTICAL AI (для Dr.Head)

### Прямые аналогии
1. **AI-система обработки заказов** (из Practical_AI_Cases.pdf) — интеграция с 1С, автораспознавание документов. Релевантно: у Dr.Head тоже 1С как основа.
   - 82% сокращение времени КП, интеграция с 1С
2. **БОТ-консьерж для отеля** — +42% рост бронирований, снижение комиссий, допродажи. Релевантно: аналогия с онлайн-продажами Dr.Head.
3. **AI-ассистент для стоматологии** — 80% автоматизация первичных обращений. Релевантно: аналогия с первой линией Dr.Head.
4. **AI-аналитика рекламных кампаний** — дашборд из 4 каналов + AI-рекомендации. Релевантно: именно это зацепило Женю (финдира).
5. **Автоматизация данных по недвижимости** — прямой бридж в CRM через API, 24/7 актуальные данные. Релевантно: задача актуализации данных из 1С.

### Метрики для Dr.Head
- 80% автоматизации первичных обращений (кейс клиники)
- +42% рост прямых продаж (кейс отеля — аналогия с онлайн)
- 82% сокращение времени на КП при интеграции с 1С
