## DARK THEME TEMPLATE

### Цвета и стили (BRAND STYLE — лаймовый, не тёмный)
```css
:root {
  --bg:       #ebebeb;   /* светло-серый фон */
  --surface:  #ffffff;   /* белые карточки */
  --lime:     #ccff00;   /* главный акцент — лайм */
  --lime-dark:#a8d400;
  --text:     #0a0a0a;   /* чёрный текст */
  --muted:    #666666;
  --border:   rgba(0,0,0,0.08);
  --red:      #e84545;   /* граница ПРОБЛЕМЫ */
  --teal:     #00a878;   /* граница РЕШЕНИЯ */
  --radius:   18px;
}
```

### Шрифт
```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700;800;900&display=swap" rel="stylesheet">
```

### Изображения-ассеты (фрукты)
- `assets/lime.png` — лайм (3D, на белом фоне) — в Hero и CTA
- `assets/raspberry.png` — малина (3D, на белом фоне) — в section-dividers

Файлы должны лежать рядом с HTML в `40_Templates/assets/` и в `docs/assets/`.

### Структура страницы (Cases Deck)

1. **TOPBAR** — бейдж "Practical AI" слева, кнопка "Сохранить PDF" справа
2. **HERO BANNER** — лаймовый фон, H1 "Кейсы Practical AI", для кого, лайм справа
3. **TASKS** — "Ваши задачи — наши решения" — 3 карточки с болями клиента (из досье)
4. **SECTION DIVIDER** — лаймовый, название категории кейсов, малина или лайм справа
5. **CASE** — белая карточка: название | ПРОБЛЕМА (красный бордер) + РЕШЕНИЕ (бирюзовый) | метрики (3 лаймовых бокса)
6. (повтор для 2-3 кейсов)
7. **HOW WE WORK** — 4 шага в 2×2 сетке, последний — лаймовый
8. **CTA** — лаймовый фон, 3 шага, контакт Данила, лайм справа
9. **FOOTER** — тёмный, лого, контакты

### Темплейт файл
`40_Templates/BRAND_TEMPLATE.html` — рабочий шаблон, копируй и заполняй плейсхолдеры.
