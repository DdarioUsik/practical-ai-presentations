1. Введение: Проблема массового спама
Крипто-обменники и крупные компании получают десятки холодных сообщений ежедневно. Они научились мгновенно распознавать шаблонные заходы и банить отправителей без раздумий.
Типичные признаки спама:
	•	Одинаковый текст для всех адресатов
	•	Общие фразы типа «мы работаем с компаниями как ваша»
	•	Отсутствие специфичных деталей о бизнесе получателя
	•	Сразу питч продукта без контекста
	•	Массовая рассылка в один день
Решение: Каждое сообщение должно показывать, что вы изучили конкретную компанию, понимаете их специфику и можете дать реальную ценность. AI помогает это масштабировать.

4. Процесс персонализированного аутрича
Пошаговый алгоритм
	•	Сегментация базы контактов — разделите контакты на группы: крипто-обменники, HR геймдев, HR корпорации, маркетинг B2B, product leads
	•	Приоритизация (5-10-50 правило) — выберите 5 топ-приоритетных (максимальная персонализация), 10 средних (средняя персонализация), 50 массовых (минимальная персонализация)
	•	AI-исследование компании — используйте промпты для Claude/Perplexity чтобы собрать: специфику бизнеса, уникальные фичи, целевую аудиторию, недавние новости/изменения
	•	Генерация персонализированного сообщения — используйте структуру: [Специфичное наблюдение] → [Релевантный кейс/опыт] → [Мягкий вопрос]
	•	Ручная проверка и редактура — убедитесь что сообщение не выглядит как AI-сгенерированное, добавьте живые детали
	•	Отправка с интервалами — не спамьте всех в один день. Разнесите по 5-10 контактов в день с интервалом 2-3 часа
	•	Трекинг и follow-up — отслеживайте ответы, через 5-7 дней делайте second touch с другим углом


5. Промпты для AI-исследования
Промпт 1: Анализ крипто-обменника
Я занимаюсь B2B продажами AI-решений для крипто-обменников. Мне нужно написать персонализированное сообщение для [название обменника].

Изучи их сайт: [URL]
Телеграм-канал (если есть): [URL]

Проанализируй и дай:
1. Какие торговые пары они поддерживают?
2. Какие регионы/страны в фокусе?
3. Какие уникальные фичи есть (P2P, API, партнерка)?
4. Есть ли проблемы/боли которые упоминают пользователи в отзывах?
5. Предложи 2-3 специфичных наблюдения для холодного сообщения
Промпт 2: Анализ HR / маркетинг компании
Я занимаюсь B2B продажами AI-автоматизации для HR-отделов. Мне нужно написать персонализированное сообщение для [имя], [должность] в [компания].

Компания: [название и краткое описание]
Индустрия: [геймдев / финтех / e-commerce / etc]

Изучи и дай:
1. Специфику их HR-процессов (объемы найма, типы позиций)
2. Текущие челленджи индустрии в рекрутинге
3. Что может быть их основной болью (скорость, качество, масштаб)?
4. Предложи специфичное наблюдение и вопрос для cold outreach
Промпт 3: Генерация персонализированного сообщения
На основе анализа создай персонализированное cold outreach сообщение:

Имя контакта: [Имя]

- Длина: 2-3 предложения

7. Автоматизация через Make.com / n8n
Сценарий автоматизации
	•	Триггер: Новая строка в Google Sheets / Airtable с контактом
	•	Scraping (опционально): Если есть URL сайта - собрать данные через Apify или Bright Data
	•	AI Research: Отправить данные в Claude API с промптом анализа
	•	Генерация сообщения: Claude API генерирует персонализированный текст
	•	Сохранение: Записать сгенерированный текст обратно в таблицу
	•	Ручная проверка: Человек проверяет и редактирует перед отправкой
	•	Отправка: После апрува - автоотправка через Telegram API / email

Пример Make.com сценария
	•	Google Sheets (Watch rows)
	•	→ HTTP Request (Claude API с промптом анализа)
	•	→ HTTP Request (Claude API для генерации сообщения)
	•	→ Google Sheets (Update row с сгенерированным текстом)
	•	→ Slack/Email (уведомление для проверки)
	•	→ Ждем апрува
	•	→ Telegram API / Email send


8. Метрики и оптимизация
Ключевые метрики
Метрика
Целевое значение
Как улучшить
Response Rate
15-25%
Больше персонализации, сильнее вопросы
Ban Rate
<5%
Убрать шаблоны, добавить уникальности
Conversation Rate
40-60% от ответов
Лучше follow-up, давать value upfront
Meeting Conversion
20-30% от диалогов
Квалифицировать лиды раньше, предлагать конкретные слоты

A/B тесты
Что тестировать:
	•	Длина сообщения (2 vs 3 предложения)
	•	Тип вопроса (про метрики vs про боли vs про процессы)
	•	Упоминание кейса (с цифрами vs без)
	•	Tone (формальный vs casual)
	•	Время отправки (утро vs день vs вечер)

Признаки что нужно менять подход
	•	Response rate < 10% — сообщения недостаточно персонализированы
	•	Ban rate > 10% — выглядите как спамер, убирайте шаблоны
	•	Отвечают но не переходят в диалог — вопросы недостаточно сильные, не зацепили боль
	•	Диалоги есть но не конвертятся в встречи — плохая квалификация лидов, предлагайте value раньше


Заключение
Персонализация через AI — это не просто автоматизация шаблонов. Это масштабирование глубокого исследования каждого потенциального клиента.
Три ключевых принципа:
	•	AI делает research, человек добавляет контекст — нейросеть собирает данные быстро, но финальная редактура критична
	•	Качество > Количество — лучше 10 персонализированных заходов чем 100 шаблонных
	•	Постоянная оптимизация — трекайте метрики, тестируйте гипотезы, улучшайте промпты

Начните с малого:
	•	Выберите 5 приоритетных контактов
	•	Потратьте 10-15 минут на каждого с AI-исследованием
	•	Создайте уникальные сообщения
	•	Отследите результаты
	•	Масштабируйте что работает
