### Портфель проектов Ильи

#### 1. Shipping Studio (AI Bootcamp)
- **Текущий статус:** 4 человека enrolled, ищет product-market fit
- **Ценообразование:** €400-1000 за участника
- **Проблема:** Участники не замотивированы, нет времени на fast-paced программу
- **Следующая итерация:** Фокус только на serious founders
- **Масштабируемость:** Может обработать 100 человек организационно, но нет аудитории

#### 2. Custom Starter Kits
- **Продано:** 1 кит за €1000 (vs €400 стандартный bootcamp)
- **Deliverable:** Pre-configured agent skills, UX specs, PRD, архитектура
- **Видение:** Продуктизировать в €200 non-custom offering
- **Корпоративный угол:** "Yandex.Chef модель" - рецепты + ингредиенты (обучение + готовое решение)

#### 3. Корпоративное AI-обучение (КОНЦЕПЦИЯ)
- **Целевая аудитория:** Middle management в корпорациях
- **Не подходит:** Top executives (делегируют), low-level сотрудники
- **Офферинг:** Vibe-coding обучение + кастомные промпты/агенты для процессов компании
- **Пример:** Кастомный генератор презентаций с брендингом компании
- **Статус:** Только идея, нет готового продукта, нет методологии, нет масштабируемой delivery
- **Timing:** Илья говорит "не хочу быть тьютором", нужно 3-4 месяца для запуска

#### 4. Mycelium Materials бизнес
- **Продукт:** Материал из мицелия для замены фанеры
- **Стадия:** Хороший R&D прогресс, small-scale production доказана
- **Нужно:** €500k-2M инвестиции для промышленного оборудования
- **Sales цель:** Chief of R&D в мебельных компаниях (пилотные партнеры, не покупатели)
- **Timeline:** Долгий (6-12 месяцев), поиск грантов
- **Недавний лид:** Большая мебельная компания в Мадриде
- **Открыт к:** Co-founder equity для бизнес-девелопмент партнера

### Что хочет Данил VS что сделал Илья

**Илья сделал:**
Telegram бот (сервис, который отвечает на сообщения пользователей)

**Данил хочет:**
Подключить Claude к Telegram, чтобы Claude **видел сообщения Данила** и помогал с задачами

**Это РАЗНЫЕ вещи.**

### Реальная потребность Данила

Судя по задаче (автоматизация sales, followup, лиды):

**Вариант 1: Telegram → Claude интеграция (Claude видит всё)**
- Данил пишет в Telegram
- Claude получает эти сообщения
- Анализирует лиды, напоминает о follow-up
- Может отвечать через бота

**Вариант 2: Экспорт Telegram → Claude (периодически)**
- Данил экспортирует чаты из Telegram
- Загружает Claude
- Claude анализирует переписки, даёт советы

**Вариант 3: Telegram бот-помощник**
- Отдельный бот в Telegram
- Данил пересылает ему сообщения от лидов
- Бот подсказывает что ответить

### Рекомендация: Вариант 2 (Экспорт) СЕЙЧАС

**Почему:**
- ✅ Быстро (5 минут)
- ✅ Без кода
- ✅ Claude сразу может проанализировать текущие лиды
- ✅ Данил поймёт нужна ли полная интеграция

**Вариант 1 (полная интеграция) ПОТОМ (через неделю)**

**Почему:**
- После закрытия R-Pharm
- Когда больше лидов
- Тогда автоматизация критична

### Инструкция: Экспорт Telegram чатов

**Шаг 1: Экспорт**
1. Открыть Telegram Desktop
2. Settings (⚙️) → Advanced → Export Telegram data
3. Выбрать:
   - ✅ Private chats (переписки с лидами)
   - ✅ Groups (если есть)
   - Format: HTML или JSON
4. Скачать файл

**Шаг 2: Загрузить Claude**

Claude проанализирует:
- Кто активные лиды
- Кому нужен follow-up
- Какие сообщения ждут ответа
- Где теряются лиды

**Шаг 3: Claude даст конкретный план**
- Кому писать завтра
- Что именно писать
- Какие лиды холодные (можно автоматизировать)

### План полной интеграции (будущее)

**Если Вариант 1 (полная интеграция):**

Нужно от Данила:
1. Создать Telegram бота через @BotFather
2. Дать Claude токен бота
3. Claude напишет скрипт который будет:
   - Читать сообщения Данила в Telegram
   - Отправлять их Claude на анализ
   - Отслеживать лиды
   - Напоминать о follow-up

### Паттерны принятия решений Данила
