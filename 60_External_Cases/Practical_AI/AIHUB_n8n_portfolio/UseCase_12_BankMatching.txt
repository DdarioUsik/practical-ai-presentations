Use Case: Подбор банка для бизнеса с помощью n8n
Интеллектуальная система Bank Matching для предпринимателей на Кипре

Бизнес-задача
Предприниматели, открывающие бизнес на Кипре, тратили недели на исследование банков и переговоры, не имея объективной информации о совместимости своего бизнес-профиля с требованиями конкретных банков. Клиенту требовалась система мгновенного подбора оптимального банковского партнёра.
Реализованное решение
На базе n8n была построена система Bank Matching, объединяющая интерактивный квиз из 8 вопросов, алгоритм скоринга по базе из 105+ банков и AI-генерацию персонализированных рекомендаций в единый автоматизированный поток.
Ключевые компоненты workflow
1. Webhook триггер — принимает данные опросника с веб-формы после прохождения квиза.
2. Модуль нормализации данных — преобразует ответы пользователя в структурированный формат для скоринга.
3. Интеграция с Airtable — сохраняет лид и загружает базу банков с характеристиками (API, мультивалюта, fast KYC).
4. Scoring Engine — рассчитывает совместимость с каждым банком по взвешенным критериям с прогрессивной фильтрацией.
5. AI-агент генерации писем — использует Google Gemini для создания персонализированных рекомендаций по топ-3 банкам.
6. Google Drive модуль — прикрепляет PDF со списком банков Кипра к письму.
7. Email-уведомления — отправляет результаты подбора через SendGrid с HTML-рассылкой.
Технические особенности
Вся логика реализована в n8n с использованием узлов Webhook, Airtable, Code для scoring engine и Google Gemini API. Кастомный JavaScript выполняет расчёт match_score по многомерной матрице критериев.
Критичным оказался дизайн прогрессивной фильтрации: количество подходящих банков сокращается после каждого ответа (105→87→62→48→32→21→12→6→1). Система классифицирует качество совпадения: ≥120 баллов — идеальное, 90-119 — хорошее, <90 — требуется ручной обзор.
Платформа обрабатывает квиз в реальном времени и отправляет персонализированное письмо с топ-3 банками в течение 10 секунд после заполнения формы.
