Use Case: Генератор инвойсов с помощью n8n
Автоматизация создания и доставки счетов через Email и Telegram
Бизнес-задача
Фрилансеры и малый бизнес тратили время на ручное создание инвойсов в редакторах, форматирование и отправку клиентам. Требовался инструмент для мгновенной генерации профессиональных счетов с автоматической доставкой на выбор клиента.
Реализованное решение
На базе n8n была построена система генерации инвойсов, объединяющая веб-интерфейс ввода данных, конвертацию HTML в PDF через внешний сервис и мультиканальную доставку (Email/Telegram) в единый автоматизированный поток.
Ключевые компоненты workflow
1. Webhook триггер — принимает JSON с данными инвойса из веб-формы (реквизиты, позиции, валюта).
2. JSON to HTML конвертер — преобразует структурированные данные в стилизованный HTML-документ инвойса.
3. Интеграция с api2pdf — отправляет HTML на внешний сервис и получает готовый PDF-файл.
4. Модуль переименования файла — присваивает PDF имя по номеру инвойса (INV-2025-001.pdf).
5. Интеграция с Airtable — сохраняет контактные данные клиента для учёта и последующих коммуникаций.
6. Switch-маршрутизатор — направляет инвойс по выбранному каналу доставки (Email или Telegram).
7. Email/Telegram доставка — отправляет PDF через SMTP или Telegram Bot API по Chat ID из базы.
Технические особенности
Вся логика реализована в n8n с использованием узлов Webhook, Code (HTML-генерация), HTTP Request (api2pdf.com), Airtable, Switch, Send Email и Telegram. Кастомный JavaScript формирует стилизованный HTML с таблицей позиций и итогами.
Критичным оказался дизайн мультиканальной доставки: для Telegram система ищет Chat ID пользователя по username в отдельной таблице Airtable, куда записываются ID после первого взаимодействия с ботом.
Frontend реализован на React 18 с Tailwind CSS и shadcn/ui, включает live-превью инвойса и валидацию форм через Zod. Среднее время от заполнения формы до получения PDF — 5 секунд.
