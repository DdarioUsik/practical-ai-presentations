Use Case: Анализ присутствия бренда в нейросетях с помощью n8n
Система мониторинга упоминаний бренда в ответах AI-систем

Бизнес-задача
Маркетологи и бренд-менеджеры не имеют инструментов для отслеживания того, как крупные языковые модели (ChatGPT, Claude, Gemini) представляют их бренд в ответах пользователям. Клиенту требовалась система для систематического анализа присутствия и тональности упоминаний бренда в генеративных AI.
Реализованное решение
На базе n8n была построена система мониторинга, объединяющая автоматический опрос нескольких AI-платформ, анализ ответов и формирование отчётов о восприятии бренда в единый автоматизированный поток.
Ключевые компоненты workflow
1. Scheduled триггер — запускает цикл опроса AI-систем по расписанию для регулярного мониторинга.
2. Модуль генерации промптов — формирует контекстные вопросы о бренде и конкурентах для AI-систем.
3. Multi-AI интеграция — отправляет запросы в ChatGPT, Claude, Gemini и другие LLM через их API.
4. Модуль анализа ответов — извлекает упоминания бренда и оценивает тональность (позитив/негатив/нейтрал).
5. Сравнительный анализ — сопоставляет позиционирование бренда относительно конкурентов.
6. Интеграция с базой данных — сохраняет результаты для построения трендов и исторического анализа.
7. Отчёт-генератор — формирует еженедельные дайджесты с ключевыми метриками и изменениями.
Технические особенности
Вся логика реализована в n8n с использованием узлов Schedule Trigger, HTTP Request (OpenAI API, Anthropic API, Google AI API), Code для анализа ответов и PostgreSQL для хранения результатов. Промпты генерируются динамически на основе заданных категорий вопросов.
Критичным оказался дизайн системы нормализации ответов: разные AI-модели используют различные форматы и стили ответов. Модуль анализа приводит все ответы к единой структуре для корректного сравнения.
Система опрашивает 5 AI-платформ по 20 промптам еженедельно, формируя базу из 100+ точек данных. Отчёты включают индекс присутствия бренда, динамику тональности и сравнение с конкурентами.
