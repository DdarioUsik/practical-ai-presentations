Use Case 05

360-оценка персонала с окном Джохари с помощью n8n

Telegram-бот для сбора обратной связи и персонального развития

Срок
3 недели
Технология
n8n workflow automation
Метрика
20 минут
Бизнес-задача
Компаниям требовался инструмент для проведения комплексной 360-оценки сотрудников, который объединяет самооценку, обратную связь от коллег и руководства, и при этом автоматически анализирует результаты. Существующие HR-платформы были дороги, сложны в настройке и не предоставляли возможности глубокого анализа через призму психологических моделей, таких как окно Джохари.
Реализованное решение
На базе n8n был построен многофункциональный Telegram-бот, который проводит комплексную 360-оценку сотрудников. Система собирает самооценку по колесу баланса и целям, распространяет анонимные опросники среди коллег и руководителей, а затем сравнивает результаты через модель окна Джохари для выявления слепых зон и скрытых сильных сторон.
Ключевые компоненты workflow
01
Telegram-бот триггер
принимает команды пользователя и запускает соответствующие сценарии оценки (самооценка, 360-ревью, модуль предназначения).
02
Модуль самооценки (SELF)
проводит пользователя через колесо баланса (8 сфер жизни), постановку целей с метриками и сроками, выбор личных качеств по чек-листу Джохари.
03
Модуль внешней оценки (OTHERS)
генерирует уникальные ссылки для анонимного опроса коллег, руководителей, подчинённых и даже близких с ролевыми вопросниками.
04
AI-аналитика окна Джохари
сравнивает выбранные качества SELF vs OTHERS и распределяет их по четырём квадрантам: открытое, слепое, скрытое, неизвестное.
05
Генератор визуальных отчётов
формирует диаграммы колеса баланса, окна Джохари и сводный Individual Development Plan.
06
AI-коуч привычек
на основе выявленных зон роста генерирует персональные ежедневные привычки и микро-эксперименты по 8 архетипам.
07
Интеграция с базой данных
сохраняет все ответы, историю оценок и прогресс пользователя для отслеживания динамики во времени.
Технические особенности
Вся логика реализована в n8n с использованием узлов Telegram, Code, HTTP Request и интеграции с AI-моделями. Кастомный JavaScript обрабатывает сложную логику сопоставления качеств по модели Джохари и расчёта баллов по биполярным шкалам. Критичным оказался дизайн модульной архитектуры: система поддерживает различные ролевые опросники (коллега, руководитель, подчинённый, партнёр, друг, ребёнок 8+), каждый со своим набором шкал и открытых вопросов. Реализована полная анонимность внешних оценок. Система автоматически рассчитывает дельту между самооценкой и внешней оценкой по 10 биполярным осям и генерирует персонализированные рекомендации. Среднее время прохождения полного цикла оценки — 15-20 минут для пользователя и 3 минуты для каждого оценщика. ___ Технология: n8n workflow automation Время внедрения: 3 недели
