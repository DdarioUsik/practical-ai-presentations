Use Case 01 · voice ai

Автоматизация входящих звонков с помощью n8n

Голосовой ИИ-ассистент для сети лучевой диагностики

Срок
11 рабочих дней
Технология
n8n workflow automation
Метрика
—
Бизнес-задача
Клиника лучевой диагностики теряла каждый третий звонок в часы пик, а содержание администраторов в ночные смены было экономически нецелесообразным. 80% обращений были типовыми: запись на КТ, уточнение цены, вопросы о подготовке.
Реализованное решение
На базе n8n построен интеллектуальный голосовой помощник, объединяющий телефонию, NLP-обработку и CRM в единый автоматизированный поток.
Ключевые компоненты workflow
01
Триггер входящего звонка
webhook от телефонного провайдера запускает n8n-процесс на первой секунде звонка.
02
Speech-to-Text модуль
преобразование речи пациента в текст с распознаванием медицинской терминологии (КЛКТ, контрастное усиление, низкодозная КТ).
03
Интеллектуальная логика обработки
ветвление диалога в зависимости от запроса: квалификация зоны исследования, информирование о противопоказаниях, проверка требований к подготовке.
04
Интеграция с YCLIENTS API
в реальном времени проверяется расписание, идентифицируются постоянные клиенты по номеру телефона, создаётся запись в журнале.
05
Text-to-Speech ответы
синтез естественной речи с медицинским контекстом для озвучивания информации пациенту.
06
SMS-уведомления
автоматическая отправка подтверждения с датой, временем и адресом клиники после успешной записи.
07
Эскалация сложных случаев
при специфических жалобах или нестандартных запросах звонок автоматически переводится на администратора с передачей контекста разговора.
Технические особенности
Вся логика реализована в n8n без написания кастомного кода. Использованы готовые узлы HTTP Request, Switch, Set, Code и встроенные интеграции с внешними API. Критичным оказался дизайн обработки ошибок: при недоступности YCLIENTS API система автоматически переключается в режим сбора заявок с последующей синхронизацией. Это обеспечило uptime 99.7%. Среднее время выполнения workflow — 3.2 секунды от входящего звонка до начала диалога.
