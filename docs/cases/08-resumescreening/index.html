Use Case 08

Скрининг резюме с помощью n8n

RAG-система автоматического анализа кандидатов для HR

Срок
10 рабочих дней
Технология
n8n workflow automation
Метрика
15 секунд
Бизнес-задача
HR-отделы крупных компаний получают сотни резюме на каждую вакансию, что делает ручной первичный скрининг трудоёмким и субъективным процессом. Клиенту требовалась система, способная автоматически оценивать соответствие кандидатов требованиям вакансии с использованием семантического анализа.
Реализованное решение
На базе n8n была построена RAG-система (Retrieval-Augmented Generation), объединяющая векторное хранилище резюме, семантический поиск и AI-оценку в единый автоматизированный поток скрининга кандидатов.
Ключевые компоненты workflow
01
Webhook триггер
принимает новые резюме из ATS (Applicant Tracking System) или email.
02
Document Parser модуль
извлекает текст из PDF/DOCX резюме и структурирует данные кандидата.
03
Embedding генератор
создаёт векторные представления резюме для семантического поиска.
04
Vector Store интеграция
сохраняет эмбеддинги в Pinecone/Qdrant для быстрого поиска похожих кандидатов.
05
RAG Query модуль
выполняет семантический поиск по требованиям вакансии в базе резюме.
06
AI-агент оценки
анализирует соответствие кандидата и формирует скоринг по критериям.
07
Результат в ATS
обновляет статус кандидата и добавляет AI-рекомендации в систему рекрутинга.
Технические особенности
Вся логика реализована в n8n с использованием узлов Webhook, Extract from File, OpenAI Embeddings, Pinecone Vector Store и GPT-4 для финальной оценки. Кастомный код выполняет нормализацию текста и chunking документов. Критичным оказался дизайн RAG-пайплайна: резюме разбиваются на семантические блоки (опыт, навыки, образование) с перекрытием для сохранения контекста. Поиск выполняется по каждому критерию вакансии отдельно с последующей агрегацией результатов. Система обрабатывает до 100 резюме в час с точностью скрининга 92% относительно экспертной оценки.
