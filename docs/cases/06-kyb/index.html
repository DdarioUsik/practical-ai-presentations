Use Case 06

KYB-оценка бизнеса с помощью n8n

Геймифицированная платформа комплаенс-проверки для предпринимателей

Срок
5 рабочих дней
Технология
n8n workflow automation
Метрика
—
Бизнес-задача
Банки и финансовые организации сталкиваются с необходимостью проводить KYB-проверки новых клиентов, что традиционно требует сложных форм и длительной обработки. Клиенту требовалась платформа, которая делает процесс оценки бизнес-рисков понятным и увлекательным для предпринимателей, одновременно собирая структурированные данные для комплаенс-анализа.
Реализованное решение
На базе n8n была построена геймифицированная платформа webankyou.com, объединяющая интерактивный опросник из 9 вопросов, систему скоринга и AI-генерацию персонализированных писем в единый автоматизированный поток оценки бизнес-рисков.
Ключевые компоненты workflow
01
Webhook триггер
принимает данные опросника с веб-формы при завершении прохождения.
02
Модуль скоринга
рассчитывает балл по 9 параметрам (юрисдикция, правовая форма, комплаенс, риски) с максимумом 108 баллов.
03
Классификатор уровня риска
распределяет клиентов по трём категориям: низкий риск (≥80), умеренный (50-79), высокий (<50).
04
Интеграция с Airtable
сохраняет данные клиента, результаты оценки и историю взаимодействий.
05
AI-агент генерации писем
использует Google Gemini для создания персонализированных рекомендаций на основе уровня риска.
06
Google Drive модуль
прикрепляет PDF чек-лист с рекомендациями к письму.
07
Email-уведомления
отправляет результаты оценки через SendGrid с персонализированным содержанием.
Технические особенности
Вся логика реализована в n8n с использованием узлов Webhook, Code, Airtable, HTTP Request (Google Gemini API), Google Drive и SendGrid. Кастомный JavaScript выполняет расчёт скоринга по взвешенным коэффициентам для каждого из 9 вопросов. Критичным оказался дизайн трёхуровневой классификации рисков: система автоматически определяет категорию клиента и выбирает соответствующий шаблон письма. AI-агент генерирует уникальные рекомендации, учитывающие конкретные ответы пользователя. Платформа поддерживает мультиязычность (EN/RU) и обрабатывает данные в реальном времени. Frontend реализован на Replit с использованием Tailwind CSS и shadcn/ui компонентов. ___ Технология: n8n workflow automation Время внедрения: 5 рабочих дней
