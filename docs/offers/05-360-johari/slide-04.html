Ключевые компоненты workflow

  • Telegram-бот триггер. принимает команды пользователя и запускает соответствующие сценарии оценки (самооценка, 360-ревью, модуль предназначения).
  • Модуль самооценки (SELF). проводит пользователя через колесо баланса (8 сфер жизни), постановку целей с метриками и сроками, выбор личных качеств по чек-листу Джохари.
  • Модуль внешней оценки (OTHERS). генерирует уникальные ссылки для анонимного опроса коллег, руководителей, подчинённых и даже близких с ролевыми вопросниками.
  • AI-аналитика окна Джохари. сравнивает выбранные качества SELF vs OTHERS и распределяет их по четырём квадрантам: открытое, слепое, скрытое, неизвестное.
  • Генератор визуальных отчётов. формирует диаграммы колеса баланса, окна Джохари и сводный Individual Development Plan.
  • AI-коуч привычек. на основе выявленных зон роста генерирует персональные ежедневные привычки и микро-эксперименты по 8 архетипам.
  • Интеграция с базой данных. сохраняет все ответы, историю оценок и прогресс пользователя для отслеживания динамики во времени.
