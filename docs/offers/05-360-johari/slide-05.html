Технические особенности

Вся логика реализована в n8n с использованием узлов Telegram, Code, HTTP Request и интеграции с AI-моделями. Кастомный JavaScript обрабатывает сложную логику сопоставления качеств по модели Джохари и расчёта баллов по биполярным шкалам. Критичным оказался дизайн модульной архитектуры: система поддерживает различные ролевые опросники (коллега, руководитель, подчинённый, партнёр, друг, ребёнок 8+), каждый со своим набором шкал и открытых вопросов. Реализована полная анонимность внешних оценок. Система автоматически рассчитывает дельту между самооценкой и внешней оценкой по 10 биполярным осям и генерирует персонализированные рекомендации. Среднее время прохождения полного цикла оценки — 15-20 минут для пользователя и 3 минуты для каждого оценщика. ___ Технология: n8n workflow automation Время внедрения: 3 недели
