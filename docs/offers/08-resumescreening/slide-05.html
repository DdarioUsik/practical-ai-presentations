Технические особенности

Вся логика реализована в n8n с использованием узлов Webhook, Extract from File, OpenAI Embeddings, Pinecone Vector Store и GPT-4 для финальной оценки. Кастомный код выполняет нормализацию текста и chunking документов. Критичным оказался дизайн RAG-пайплайна: резюме разбиваются на семантические блоки (опыт, навыки, образование) с перекрытием для сохранения контекста. Поиск выполняется по каждому критерию вакансии отдельно с последующей агрегацией результатов. Система обрабатывает до 100 резюме в час с точностью скрининга 92% относительно экспертной оценки. Среднее время обработки одного резюме — 15 секунд. ___ Технология: n8n workflow automation Время внедрения: 10 рабочих дней
