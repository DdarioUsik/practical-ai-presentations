Ключевые компоненты workflow

  • Webhook триггер. принимает новые резюме из ATS (Applicant Tracking System) или email.
  • Document Parser модуль. извлекает текст из PDF/DOCX резюме и структурирует данные кандидата.
  • Embedding генератор. создаёт векторные представления резюме для семантического поиска.
  • Vector Store интеграция. сохраняет эмбеддинги в Pinecone/Qdrant для быстрого поиска похожих кандидатов.
  • RAG Query модуль. выполняет семантический поиск по требованиям вакансии в базе резюме.
  • AI-агент оценки. анализирует соответствие кандидата и формирует скоринг по критериям.
  • Результат в ATS. обновляет статус кандидата и добавляет AI-рекомендации в систему рекрутинга.
Интеграции
  • Webhook / входящие события
  • Парсинг документов (PDF/DOCX), OCR при необходимости
  • LLM/Embeddings, векторное хранилище (если нужно)
  • CRM/ATS/BI — запись результата обратно
