Ключевые компоненты workflow

  • Webhook триггер. принимает данные опросника с веб-формы при завершении прохождения.
  • Модуль скоринга. рассчитывает балл по 9 параметрам (юрисдикция, правовая форма, комплаенс, риски) с максимумом 108 баллов.
  • Классификатор уровня риска. распределяет клиентов по трём категориям: низкий риск (≥80), умеренный (50-79), высокий (<50).
  • Интеграция с Airtable. сохраняет данные клиента, результаты оценки и историю взаимодействий.
  • AI-агент генерации писем. использует Google Gemini для создания персонализированных рекомендаций на основе уровня риска.
  • Google Drive модуль. прикрепляет PDF чек-лист с рекомендациями к письму.
  • Email-уведомления. отправляет результаты оценки через SendGrid с персонализированным содержанием.
Интеграции
